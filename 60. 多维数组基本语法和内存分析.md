# 1. 二维数组举例：
```
int [][] a = {{1,2},{3,4,0,9},{5,6,7}};    （数组元素还是数组）
```

* Java中多维数组不必是规则矩阵形式。

| 	|j=0 |	j=1 |	j=2 |	j=3|

|i=1 |	3 |	4 |	0 |	9 |
|i=2 |	5 |	6 |	7 |	  |

# 3. Java中多维数组的声明和初始化
应按从 **高维到低维** 的顺序进行，例如：
```
int [][] a = new int [3][];   //定义一级数组长度为3
a[0] = new int[2];  //二级数组长度为2
a[1] = new int[4];  //二级数组长度为4
a[2] = new int[3];  //二级数组长度为3
int t1[][] = new int[][4]; //非法
```
